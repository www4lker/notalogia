# teclado com layout internacional em português

## criado em: 
-  25-04-2023 - 12:29
#TIPS #ANELO 
---

Aqui está um cheat sheet útil para você usar seu teclado com layout internacional em português:

1. Acentos:
   - Á: `AltGr + A`
   - É: `AltGr + E`
   - Í: `AltGr + I`
   - Ó: `AltGr + O`
   - Ú: `AltGr + U`
   - Â: `AltGr + Shift + A`
   - Ê: `AltGr + Shift + E`
   - Î: `AltGr + Shift + I`
   - Ô: `AltGr + Shift + O`
   - Û: `AltGr + Shift + U`
   - À: `AltGr + ` (tecla ao lado do Enter) + A`
   - È: `AltGr + ` (tecla ao lado do Enter) + E`
   - Ì: `AltGr + ` (tecla ao lado do Enter) + I`
   - Ò: `AltGr + ` (tecla ao lado do Enter) + O`
   - Ù: `AltGr + ` (tecla ao lado do Enter) + U`
   - Ç: `AltGr + , (vírgula)`

2. C cedilha:
   - ç: `AltGr + , (vírgula)`

3. Sinais de pontuação:
   - !: `Shift + 1`
   - ?: `Shift + / (barra)`
   - ;: `Shift + , (vírgula)`
   - : `Shift + . (ponto)`
   - -: `- (hífen)`
   - _: `_ (underline)`
   - ": `AltGr + Shift + ' (aspas)`

4. Outros caracteres:
   - º: `AltGr + 0 (zero)`
   - ª: `AltGr + 1`
   - €: `AltGr + E`
   - @: `AltGr + 2`
   - # (jogo da velha): `AltGr + 3`
   - $: `AltGr + 4`
   - %: `AltGr + 5`
   - & (e comercial): `AltGr + 6`
   - *: `AltGr + 8`
   - ( e ): `Shift + 8 e 9`
